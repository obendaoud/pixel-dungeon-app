# References

Toutes les ressources externes du curriculum, classées par type.

## Fichiers

- `channels.md` : chaînes YouTube et créateurs à suivre, classés par phase et par auteur
- `books.md` : livres de référence
- `tools.md` : outils logiciels (Aseprite, Godot, plugins, palettes)
- `palettes.md` : palettes recommandées et liens Lospec
- `frame-references.md` : sources de frames anime / KOF / pixel art à étudier en frame-by-frame

## Comment utiliser

Avant chaque leçon, regarde la section `Démonstration` de la leçon pour identifier les vidéos / liens spécifiques à étudier. Puis reviens ici si tu veux explorer plus largement un auteur ou un sujet.

## Note sur les liens YouTube

Les leçons font référence à des vidéos par leur titre + nom de la chaîne, sans URL fixe (les URLs YouTube changent et les vidéos sont parfois supprimées). Tu fais une recherche sur YouTube avec le titre exact + le nom de la chaîne.

Si une vidéo n'existe plus, cherche une vidéo équivalente du même auteur ou d'un autre auteur reconnu sur le même sujet. Le contenu théorique des leçons est suffisant pour avancer même sans la vidéo précise.
