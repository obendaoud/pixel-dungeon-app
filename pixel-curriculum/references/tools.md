# Outils

## Aseprite

Logiciel principal du curriculum. Tout est conçu autour.

- **URL** : aseprite.org
- **Prix** : ~20€ (paiement unique)
- **OS** : Windows, macOS, Linux
- **Version requise** : 1.3 ou supérieure (pour Tilesets et Slices)
- **Open source** : oui (compilation manuelle gratuite, mais build officiel payant)

## Godot 4

Moteur cible de Monsieur Obo (phase 4).

- **URL** : godotengine.org
- **Prix** : gratuit, open source, MIT license
- **OS** : Windows, macOS, Linux, Web
- **Version requise** : Godot 4.0+ (Godot 4.2+ recommandé pour les améliorations 2D)

### Plugins Godot recommandés

**Aseprite Wizard**
- URL : github.com/viniciusgerevini/godot-aseprite-wizard
- AssetLib : "Aseprite Wizard"
- Pertinence : Phase 4.9
- Fonction : import automatique des `.aseprite` vers SpriteFrames Godot, avec tags = animations

**Pixel Pro Tools**
- AssetLib : rechercher "Pixel" pour les outils 2D
- Plusieurs petits outils utiles : pixel snap helpers, sprite outline shaders, etc.

## Lospec

Site web pour palettes pixel art.

- **URL** : lospec.com
- **Section principale** : `lospec.com/palette-list`
- **Format export** : `.gpl`, `.hex`, `.png`, `.aseprite-palette`
- **Pertinence** : Phase 1.5, 2.4, 5.1

### Palettes recommandées sur Lospec

- **Game Boy** : 4 couleurs, exercices phase 1
- **Sweetie 16** : 16 couleurs, recommandée pour apprentissage Phase 2
- **Resurrect 32** : 32 couleurs équilibrées, alternative phase 5
- **Endesga 32** : 32 couleurs, palette de référence game art
- **Endesga 64** : 64 couleurs, plus large, phase 5
- **NA16** : 16 couleurs, alternative à Sweetie 16
- **Pico-8** : 16 couleurs, palette ultra connue
- **AAP-64** : 64 couleurs, équilibrée

## Captureurs et exporters (LinkedIn / vidéo)

**OBS Studio**
- URL : obsproject.com
- Gratuit, open source
- Pertinence : pour capturer des timelapses Aseprite (recording session) et des démos Godot
- Note : configurer en H.264, 60 fps, résolution native pour préserver le pixel art

**ScreenToGif**
- URL : screentogif.com
- Gratuit, Windows uniquement
- Pertinence : capturer des clips courts (5-15s) en GIF, parfait pour LinkedIn

**ezgif.com**
- URL : ezgif.com
- Web, gratuit
- Pertinence : convertir / découper / optimiser des GIF en ligne. Utile pour préparer le contenu social.

## Outils complémentaires

**Pyxel Edit** (alternative à Aseprite)
- URL : pyxeledit.com
- Pertinence : alternative payante à Aseprite. Pas couvert par le curriculum, à éviter pour simplicité.

**Piskel** (alternative gratuite et web)
- URL : piskelapp.com
- Pertinence : alternative gratuite, browser-based. Moins puissant qu'Aseprite. Pas recommandé pour le curriculum mais utile en dépannage.

**GIMP**
- URL : gimp.org
- Gratuit, open source
- Pertinence : pas pour le pixel art en soi, mais utile pour des manipulations d'image (composition de sheets, post-traitement). Configurer en pixel mode si utilisé.
