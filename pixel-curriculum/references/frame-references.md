# Frame references

Sources de frames à étudier en frame-by-frame pour comprendre comment les pros animent. Ces sources sont à observer, pas à copier.

## Anime shonen (frames d'animation TV)

À étudier pour comprendre l'anticipation forte, les smear frames, les impact frames.

### Jujutsu Kaisen (JJK)

- Saison 1, épisode 9 : combat Itadori vs Junpei (mouvement basique, anticipation classique)
- Saison 1, épisode 19 : Yuji Itadori "Divergent Fist" (frame breakdown : anticipation forte, smear, impact, hit stop dramatique)
- Saison 2 : combat Mahito vs Itadori (références d'animation MAPPA en haute qualité, smear et impact très lisibles)
- Movie 0 : combat Yuta vs Geto (animation cinéma, encore plus poussée)

### Demon Slayer (Kimetsu no Yaiba)

- Saison 1, épisode 19 : Hinokami Kagura (la séquence iconique)
- Saison 1, épisode 26 : combat final Tanjiro vs Rui (référence absolue de combat anime moderne)
- Mugen Train movie : nombreuses scènes de combat
- Saison 3 (Swordsmith Village) : combat Hashira

### Bleach (Thousand-Year Blood War)

- Combats Ichigo vs Yhwach (référence pour le combat tournoyant)

### My Hero Academia

- Combats Deku vs Muscular, Deku vs Bakugo final
- Animation par Studio Bones

### Méthode d'étude

1. Trouve la scène sur YouTube ou ton service de streaming
2. Mets en pause sur l'anticipation
3. Avance frame par frame avec la touche `,` ou `.` (selon le player) ou utilise un site comme animatedscreenshots.com
4. Identifie les frames :
   - Anticipation (combien de frames, quelle pose, hold visible ?)
   - Action / smear (combien de frames, déformation visible ?)
   - Impact (white flash, cut frame, ou frame normale ?)
   - Hit stop (combien de frames identiques tenues ?)
   - Recovery
5. Note les patterns. Tu vas voir que les frames d'anticipation longues sont SYSTÉMATIQUES sur les coups importants.

## Jeux fighting 2D

Pour étudier les frames de combat dans un contexte jeu (donc encore plus codifié).

### KOF XIII (King of Fighters)

- 100% pixel art professionnel
- Animations de tous les characters disponibles en frame data online
- Référence absolue pour les supers et les coups normaux
- Site avec frame data : dreamcancel.com pour KOF XIII

### Garou: Mark of the Wolves

- SNK, 1999. Pixel art classique
- Animations très fluides, smear frames visibles
- Étudier les supers de Rock Howard, Terry, Kim Dong Hwan

### Street Fighter III: Third Strike

- Capcom, 1999. Pixel art ultra-fluide
- Référence pour le style classique des fighting games 2D
- Étudier les normales et les supers

### Skullgirls

- Vectoriel mais inspiration anime forte
- Frame data disponible sur le wiki du jeu
- Bonnes références pour les smear frames stylisés

### Méthode d'étude

1. Cherche "frame data" pour le jeu (ex : "KOF XIII frame data")
2. Tu obtiens les sprites sheets et le timing de chaque attaque
3. Étudie les sheets : combien de frames d'anticipation, combien d'action, combien de recovery
4. Cherche aussi des vidéos de gameplay au ralenti (slow-mo) sur YouTube

## Films / animation traditionnelle

Pour les principes de base, surtout walk cycles et pose dynamique.

### Studio Ghibli

- Princesse Mononoké : combat Ashitaka vs Demon (tout est dans la posture)
- Ponyo : animation aquatique
- Nausicaä : animation de vol et de combat aérien

### Films animation Disney classiques

- The Lion King : runs et chases
- Mulan : combat dynamique avec armes
- Tarzan : mouvements de balancement

### Spider-Man: Into the Spider-Verse / Across the Spider-Verse

- Animation hybride 2D / 3D mais frame timing très étudié
- Référence pour le smear frame moderne
- Hit stop subtil mais présent

## Tools pour frame study

- **YouTube** : touche `,` et `.` pour avancer / reculer frame par frame en pause
- **VLC** : bouton "frame by frame" en pause (Shift+E par défaut)
- **Aseprite** : tu peux importer un MP4 (avec ffmpeg en backend), le découper en frames PNG, et les charger comme reference layers pour traceback
- **animatedscreenshots.com** ou similaires : pour des captures frame-by-frame en image
- **CapCut / DaVinci Resolve** : montage vidéo gratuits qui permettent slow-mo et frame-by-frame

## Note éthique

Étudier des frames d'oeuvres existantes pour apprendre est légitime. Reproduire trait pour trait pour publier sous ton nom ne l'est pas. Inspire-toi, comprends, mais re-dessine de zéro avec ta propre vision.
