# Livres de référence

Les livres ne sont pas obligatoires. Ils servent à approfondir si une leçon te laisse sur ta faim. Ils sont en anglais sauf mention contraire.

## Animation

**The Illusion of Life: Disney Animation**
- Auteurs : Frank Thomas, Ollie Johnston
- Publication : 1981
- Pertinence : Phase 3
- Pourquoi : Le livre fondateur des 12 principes de l'animation. Toute personne qui anime sérieusement devrait l'avoir au moins consulté.
- Note : Long et cher. Une lecture en bibliothèque ou achat d'occasion suffit.

**The Animator's Survival Kit**
- Auteur : Richard Williams
- Publication : 2001
- Pertinence : Phase 3, 5
- Pourquoi : Manuel pratique de l'animation, par le directeur d'animation de "Roger Rabbit". Couvre walk cycles, run cycles, dialogue, action.
- Note : Plus pratique et moins théorique que "The Illusion of Life". Très bon investissement.

**The Animator's Survival Kit (Expanded Edition)**
- Version étendue avec DVD d'exemples animés. Si tu peux trouver, prends celle-ci.

## Anatomie pour animation

**Force: Dynamic Life Drawing for Animators**
- Auteur : Mike Mattesi
- Publication : 2006 (édition Expanded en 2017)
- Pertinence : Phase 5
- Pourquoi : Référence absolue sur la line of action et le gesture drawing pour animateurs. Indispensable pour la phase 5.

**Drawing the Head and Hands**
- Auteur : Andrew Loomis
- Publication : 1956
- Pertinence : Phase 5
- Pourquoi : Classique sur l'anatomie de la tête et des mains. Domaine public, téléchargeable légalement.

**Figure Drawing for All It's Worth**
- Auteur : Andrew Loomis
- Publication : 1943
- Pertinence : Phase 5
- Pourquoi : Anatomie complète pour le dessin. Domaine public, téléchargeable.

## Pixel art (peu de livres)

Le pixel art a peu de livres canoniques. La majorité de la connaissance vit en ligne (YouTube, Twitter, Patreon, forums). Les références :

**Make Your Own Pixel Art**
- Auteur : Jennifer Dawe, Matthew Humphries
- Publication : 2019
- Pertinence : Phase 1, 2
- Pourquoi : Manuel pratique pixel art pour débutants. Couvre Aseprite, palette, lumière. Introduction solide.

**The PixelArt Bible**
- Auteur : divers, compilation
- Publication : variable
- Note : Plusieurs livres avec des titres similaires existent. Vérifier les reviews avant d'acheter.

## Color theory

**Color and Light: A Guide for the Realist Painter**
- Auteur : James Gurney
- Publication : 2010
- Pertinence : Phase 2 (avancé)
- Pourquoi : Théorie complète de la lumière et de la couleur, applicable au pixel art via les principes (hue shifting, lumière indirecte, reflected light).

## Game design (contexte)

**The Art of Game Design: A Book of Lenses**
- Auteur : Jesse Schell
- Publication : 2008
- Pertinence : Tangentiel
- Pourquoi : Pas pixel art, mais utile pour comprendre comment tes sprites s'inscrivent dans une expérience de jeu.

## Combat animation et anime

Pas de livres canoniques sur le sujet. La connaissance se trouve principalement en analysant les frames d'anime et de jeux fighting. Voir `frame-references.md`.
